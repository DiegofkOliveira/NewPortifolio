ferramentas

Tecnologias que impulsionam meu trabalho

  • Html

    É a base de toda página web. Estruturando elementos como textos, imagens e links. Essencial para criar layouts funcionais e desenvolver sites e aplicativos.

  • Css

    CSS é responsável pela estilização de páginas web, definindo cores, fontes e layouts. Essencial para criar designs atraentes e responsivos.

  • Javascript

    JavaScript permite interatividade em sites, como animações, validações e dinâmicas em tempo real. Fundamental para aplicações modernas.

  • Figma

    Figma é uma ferramenta de design colaborativa para criar protótipos e interfaces, permitindo feedback e ajustes em tempo real.

  • Git

    Git é um sistema de controle de versão que gerencia alterações no código, garantindo organização e colaboração eficiente.

  • Vite.js

    Vite.js é um bundler moderno e rápido, projetado para melhorar a experiência de desenvolvimento, com suporte a hot reloading e otimização de desempenho.
